## Готовые программы

Проверяемые примеры находятся в папке [examples](examples/):

- [variables.shn](examples/variables.shn) — переменные, типы и константы;
- [functions.shn](examples/functions.shn) — функции и возвращаемые значения;
- [loops.shn](examples/loops.shn) — все формы цикла;
- [lists.shn](examples/lists.shn) — методы и агрегаты списков;
- [strings.shn](examples/strings.shn) — строки и интерполяция;
- [mathematics.shn](examples/mathematics.shn) — математические функции;
- [data-report.shn](examples/data-report.shn) — небольшая законченная программа.

Запуск примера:

```bash
shine run docs/examples/data-report.shn
```

Проверка без выполнения:

```bash
shine check docs/examples/data-report.shn
```

## Обозначения

Блоки с пометкой `shine` содержат код языка:

```shine
fn main() {
    print("Hello, Shine")
}
```

Команды терминала помечены как `bash`. Строки с комментарием `// ошибка` в документации показывают намеренно неверный код и не предназначены для запуска.

> Важно: из-за конфликта синтаксиса `//` в текущем MVP комментарий должен занимать отдельную логическую строку. Внутри выражения `//` означает целочисленное деление.
