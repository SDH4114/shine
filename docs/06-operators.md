# Операторы и выражения

## Арифметика

| Оператор | Значение | Пример |
|---|---|---|
| `+` | сложение | `a + b` |
| `-` | вычитание | `a - b` |
| `*` | умножение | `a * b` |
| `/` | деление с результатом `Float` | `a / b` |
| `//` | целочисленное деление | `a // b` |
| `%` | остаток | `a % b` |
| `**` | степень | `a ** b` |

```shine
print(7 + 3)
print(7 - 3)
print(7 * 3)
print(7 / 3)
print(7 // 3)
print(7 % 3)
print(2 ** 8)
```

`/` всегда возвращает `Float`. `**` также возвращает `Float`.

Для двух `Int` оператор `//` возвращает целую часть частного с округлением к нулю. Если участвует `Float`, результат округляется вниз и возвращается как `Float`.

Деление или остаток по модулю нуля вызывает `Math Error`.

## Унарные операторы

```shine
negative = -10
positive = +10
opposite = not true
```

`+` и `-` требуют число. `not` преобразует истинность значения в противоположный `Bool`.

## Сравнения

| Оператор | Значение |
|---|---|
| `==` | равно |
| `!=` | не равно |
| `<` | меньше |
| `<=` | меньше или равно |
| `>` | больше |
| `>=` | больше или равно |

```shine
print(10 == 10)
print("A" != "B")
print(3.5 <= 4)
```

Числа `Int` и `Float` можно сравнивать между собой. Строки сравниваются лексикографически. Для порядка используйте совместимые типы.

Цепочки сравнений пока не являются отдельной возможностью языка. Вместо `0 < x < 10` пишите:

```shine
0 < x and x < 10
```

## Логика

```shine
if age >= 18 and hasTicket {
    print("Allowed")
}

if isAdmin or isOwner {
    print("Access granted")
}

if not finished {
    print("Still working")
}
```

`and` и `or` используют короткое замыкание:

- `a and b` не вычисляет `b`, если `a` ложно;
- `a or b` не вычисляет `b`, если `a` истинно.

Результат логических операторов — `Bool`.

## Истинность значений

Ложными считаются:

- `false`;
- `none`;
- `0`;
- `0.0`;
- пустая строка `""`;
- пустой список `[]`;
- пустой диапазон.

Остальные поддерживаемые значения истинны. Но равенство остаётся строгим по смыслу: `false != 0`.

## Принадлежность

```shine
names = ["Amin", "Murad"]

if "Amin" in names {
    print("Found")
}

if names.have("Murad") {
    print("Also found")
}

if "ine" in "Shine" {
    print("Substring found")
}
```

`in` возвращает `Bool`. Для списка он ищет равное значение, для строки — подстроку.

## Строки и списки

```shine
fullName = "Shine" + " Language"
line = "-" * 20

first = [1, 2]
second = [3, 4]
combined = first + second
repeated = [0, 1] * 3
```

Смешивать строку и число через `+` нельзя. Используйте интерполяцию или `string(value)`.

## Приоритет операторов

От высокого приоритета к низкому:

1. вызов, метод, индекс, срез: `f()`, `x.m()`, `x[i]`;
2. степень `**` — правоассоциативна;
3. унарные `-`, `+`, `not`;
4. `*`, `/`, `//`, `%`;
5. `+`, `-`;
6. `<`, `<=`, `>`, `>=`, `in`;
7. `==`, `!=`;
8. `and`;
9. `or`;
10. диапазон `..`.

Используйте скобки, когда порядок должен быть очевиден:

```shine
result = (a + b) * c
condition = ready and (count > 0 or forced)
```
