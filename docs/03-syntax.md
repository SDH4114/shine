# Основы синтаксиса

## Файлы и кодировка

Исходные файлы Shine используют расширение `.shn`. Имена переменных и функций могут содержать Unicode-буквы, цифры и `_`, но имя не может начинаться с цифры.

```shine
name = "Shine"
user_name = "Amin"
число = 10
```

## Блоки

Блоки заключаются в фигурные скобки. Отступы улучшают читаемость, но не меняют грамматику.

```shine
if temperature > 30 {
    print("Hot")
}
```

Стандартный отступ formatter — четыре пробела.

## Разделение инструкций

Обычно каждая инструкция записывается на отдельной строке:

```shine
x = 10
y = 20
print(x + y)
```

Точка с запятой необязательна. Она позволяет разделить инструкции на одной строке:

```shine
x = 10; y = 20; print(x + y)
```

Formatter удаляет завершающие точки с запятой и предпочитает отдельные строки.

## Комментарии

Комментарий начинается с `//`, когда это первый значимый элемент строки:

```shine
// Calculate the circle area.
area = PI * radius ** 2
```

В текущей версии не помещайте комментарий после выражения:

```shine
// Правильно
result = 10 // 3
```

Внутри выражения `//` — оператор целочисленного деления. Поэтому комментарии следует писать на отдельных строках.

## Литералы

```shine
42
19.99
3e8
1.5e-10
8_000_000
"Hello"
true
false
none
[1, 2, 3]
```

Подчёркивания в числах игнорируются и могут использоваться для читаемости:

```shine
population = 8_000_000_000
avogadro = 6.022e23
```

## Ключевые слова

Зарезервированные слова:

```text
fn const if else loop in step return break continue
true false none and or not
```

Их нельзя использовать как имена переменных или функций.

## Регистр

Shine чувствителен к регистру:

```shine
name = "Amin"
Name = "Murad"
```

Это две разные переменные. Названия встроенных типов тоже чувствительны к регистру: `Int` правильно, `int` — неизвестный тип.

## Главная функция

Обычная консольная программа определяет `main`:

```shine
fn main() {
    print("Program started")
}
```

Объявления верхнего уровня выполняются до `main`. Функции рекомендуется определять только на верхнем уровне.

## Полная минимальная программа

```shine
const GREETING: String = "Hello"

fn greet(name: String): String {
    return "{GREETING}, {name}!"
}

fn main() {
    user = "Shine programmer"
    print(greet(user))
}
```
